This dashboard is powered by a delta-based data pipeline. A Python script extracts new data daily via API, appends only the new records to a master dataset stored on a network drive.

Power BI is used to transform and model the data using advanced Power Query techniques and DAX logic. This dashboard aims to uncover insights and tell compelling stories through clean design, intuitive visuals, and creative use of Power BI's native features.
